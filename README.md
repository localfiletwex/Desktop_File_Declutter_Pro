# 🧹 Desktop File Declutter Engine Pro

A lightweight, completely local Windows utility built in Python to instantly rescue your chaotic `Downloads` or `Desktop` folders. Reorganize hundreds of files in a single click by file extension or file age, with a built-in state engine to completely reverse the process if you change your mind.

## ✨ Features
* **Folder Picker:** Tap "Browse..." to target any drive, directory, or folder on your system dynamically.
* **Sort by File Type:** Automatically groups extensions into dedicated folders (`/Documents`, `/Images`, `/Code_and_Scripts`, etc.).
* **Sort by File Age:** Instantly groups files based on their last modified timestamps (`/Today`, `/This_Week`, `/This_Month`, `/Older_Than_A_Month`).
* **Instant Safety Undo:** Changed your mind? Hit the Undo button to read the execution log and snap all items back to their exact original homes.
* **100% Private:** Operates entirely offline on your hard drive. Zero tracking, zero telemetry, and zero data collecting.
